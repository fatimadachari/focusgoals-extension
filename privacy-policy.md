# Política de Privacidade - FocusGoals

Última atualização: 03/02/2026

## Coleta de Dados
A extensão FocusGoals NÃO coleta, armazena ou transmite nenhum dado pessoal para servidores externos.

## Armazenamento Local
Todos os dados (configurações, estatísticas, histórico) são armazenados APENAS localmente no seu navegador usando Chrome Storage API.

## Permissões Necessárias
- **storage**: Para salvar suas configurações e estatísticas
- **alarms**: Para gerenciar o timer Pomodoro
- **notifications**: Para alertas quando sessões terminam
- **tabs**: Para bloquear sites durante foco
- **host_permissions**: Para detectar sites bloqueados

## Compartilhamento de Dados
Nenhum dado é compartilhado com terceiros. Tudo permanece no seu dispositivo.
