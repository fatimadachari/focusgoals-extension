Política de Privacidade - FocusGoals

Última atualização: 03 de fevereiro de 2026

Coleta de Dados

A extensão FocusGoals NÃO coleta, armazena ou transmite nenhum dado pessoal para servidores externos.

Armazenamento Local

Todos os dados (configurações, estatísticas, histórico de produtividade) são armazenados APENAS localmente no seu navegador usando a Chrome Storage API. Nenhuma informação sai do seu dispositivo.

Permissões Necessárias

A extensão solicita as seguintes permissões para funcionar:

  • storage: Para salvar suas configurações, estatísticas e histórico de produtividade localmente no navegador
  • alarms: Para gerenciar o timer Pomodoro e executar ações em segundo plano (como notificações)
  • notifications: Para exibir alertas quando sessões de foco ou pausa terminam
  • tabs: Para detectar URLs atuais e bloquear sites configurados durante sessões de foco
  • host_permissions (<all_urls>): Para verificar a URL da aba atual e exibir indicador de foco em qualquer página

Compartilhamento de Dados

Nenhum dado é compartilhado com terceiros. A extensão não se comunica com nenhum servidor externo. Todos os seus dados permanecem exclusivamente no seu dispositivo.

Segurança

Como todos os dados são armazenados localmente no seu navegador, a segurança dos dados depende da segurança do seu dispositivo e navegador.

Alterações nesta Política

Quaisquer alterações nesta política de privacidade serão publicadas nesta página. Recomendamos que você revise periodicamente.

Código Aberto

FocusGoals é um projeto open source. Você pode revisar o código fonte completo no GitHub.
